# 计算生物学课程大作业要求 — 空间转录组分析 Agent

## 一、框架要求（§1）

1. 节点设计：至少包含任务理解、工具调用、错误处理、结果解释四类节点
2. 边与流程控制：支持条件路由、循环执行
3. 状态管理：定义 Agent 状态 TypedDict，使用 Reducer 机制合并状态
4. 循环机制：支持步骤级的执行-检查-修正循环

## 二、结构要求（§2）

1. Prompt 设计：包含角色定义、工具描述、约束规则
2. Tool 设计：每个工具需有签名 sig、描述 desc 和可调用函数 fn
3. State / Memory：跨轮状态传递和对话记忆
4. Skill / Workflow：预定义命名分析工作流
5. Loop 控制：条件循环和终止条件

## 三、功能要求（§3）

基础功能（8 项）：自然语言输入、数据读取（.h5ad）、数据概览、质量控制、预处理（归一化/HVG/PCA）、聚类（UMAP + Leiden）、空间可视化、结果解释

扩展功能（6 项，至少实现 3 项）：
- 空间邻域分析
- SVG 识别（Moran's I）
- 细胞通讯（配体-受体）
- 自动报告生成
- 多数据类型支持
- 前端交互增强
